== Installation ==

**Quick & Painless Setup**

1. Install & activate **Bizuno Payments** through the WordPress Plugins screen (or upload from GitHub).
2. Go to **WooCommerce ‚Üí Settings ‚Üí Integrations** (or Bizuno settings if using the full Bizuno portal plugin).
3. Enter your Bizuno Payment credentials.
5. Save

For full Bizuno self-hosted users: activate alongside the Bizuno Accounting portal plugin for the complete experience.
